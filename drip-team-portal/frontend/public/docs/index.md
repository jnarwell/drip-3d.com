# Drip Team Portal

Drip Team Portal is an engineering validation and tracking system for hardware R&D teams.

## Core Sections

### Dashboard
System validation progress, test protocol status, campaign charts, risk assessment, and recent activity.

### Components
Component registry with properties, materials integration, expression-based calculations, and status tracking.

### Models
Physics model templates with inputs, outputs, equations, and dimensional analysis.

### Analysis
Named model instances with bound inputs, real-time WebSocket updates, and chained calculations.

### Testing
Test protocols with input/output parameters, run execution, measurement recording, and validation.

### Time Tracking
Timer and manual entry, break tracking, Linear issue integration, team view with stats.

### Reports
Dashboard stats, charts, activity feed, feedback triage, and export functionality.

### Resources
Property tables (37 tables), constants, documents with Google Drive integration, and contacts.

### Settings
Theme preferences, unit settings across 30+ dimensions, and password management.
