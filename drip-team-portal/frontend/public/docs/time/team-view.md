# Team View

View aggregated time data across all team members.

## Features

### Active Timers Sidebar
- Green dot = timer running
- Yellow dot = on break

### Team Stats Row
- Total hours
- Entries count
- Average per person
- % of target

### Charts
- Project breakdown bar chart
- Issue breakdown bar chart

## Entry List

- Grouped by date (descending)
- Shows: time range, duration, breaks, categorization
- Edit/Delete buttons

## Data Access

All authenticated users can view team time data.
