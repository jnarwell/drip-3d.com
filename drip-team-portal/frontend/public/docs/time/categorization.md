# Categorization

Required when stopping timer or submitting manual entry.

## Linking Options

### Linear Issue
- Auto-fetches active issues
- Search by title or ID

### Resource/Document
- Title
- Type
- Optional URL

### N/A
Mark as uncategorized.

## Description

Optional text field for notes.

## Edit Entry

Entries can be edited after creation.

### Requirements
- Edit reason selection required
- Full edit history maintained

## Gotchas

- Edit reason required to save edits
- Full edit history maintained
