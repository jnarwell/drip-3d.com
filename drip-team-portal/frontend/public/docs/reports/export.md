# Export

Export validation reports and data.

## Available Exports

### Validation Report (PDF)
- Component verification status
- Test protocol execution results
- Physics model validation data
- Date-stamped for selected range

### Test Campaign Data (Excel)
Sheets for:
- Components
- Test Protocols
- Test Runs
- Time Entries
- Physics Validation

## Downloading

1. Select date range
2. Click "Download PDF" or "Download Excel"
3. Button shows "Generating..."
4. File downloads automatically

## Date Range

Filters:
- Test runs within range
- Time entries within range
- Activity within range

Components and protocols always included fully.
