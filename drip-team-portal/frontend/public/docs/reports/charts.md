# Charts

## Component Status Pie Chart

Breakdown by verification status:
- NOT_TESTED (gray)
- IN_TESTING (blue)
- VERIFIED (green)
- FAILED (red)

## Hours by Team Member Bar Chart

Horizontal bars showing hours per person for the date range.
- Sorted by hours (highest first)

## Test Campaign Progress Line Chart

- Solid green line: Completed tests (cumulative)
- Dashed blue line: Planned tests (target)

## Stats Cards

| Card | Value | Subtitle |
|------|-------|----------|
| Components | Total count | Failed count |
| Verified | Verified count | Percentage |
| Tests Complete | Completed count | "of X total" |
| Hours Logged | Total hours | Hours and minutes |

## Auto-Refresh

Data refreshes every 60 seconds.
