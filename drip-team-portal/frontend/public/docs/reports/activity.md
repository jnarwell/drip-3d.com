# Activity

Recent Activity feed shows last 10 activities.

## Activity Entry

| Element | Description |
|---------|-------------|
| Icon | Type indicator |
| Action | What happened |
| User | Who performed action |
| Timestamp | When it occurred |
| Type Badge | Activity category |

## Activity Types

- component: Created, updated, deleted
- test: Started, completed, aborted
- protocol: Created, updated, deactivated
- resource: Uploaded, updated, deleted
- time_entry: Created, categorized

## Action Colors

| Action | Color |
|--------|-------|
| created | Green |
| updated | Blue |
| deleted | Red |

## Auto-Refresh

Refreshes every 60 seconds with page data.
