# Reports

**Route:** `/reports`

## Dashboard

- **Auto-refresh**: Every 60 seconds
- **Date range filtering**: Week, Month, Quarter, Year, Custom

## Stats Cards

| Card | Description |
|------|-------------|
| Total Components | Component count |
| Components Verified | Verified count and percentage |
| Tests Complete | Completed vs total |
| Hours Logged | Total hours for date range |

## Charts

- **Component Status pie chart**: Breakdown by status
- **Hours by Team Member bar chart**: Hours per person
- **Test Campaign Progress line chart**: Completed vs planned over time

## Recent Activity Feed

Last 10 activities across the platform.

## Resources Overview

Summary of resource statistics.
