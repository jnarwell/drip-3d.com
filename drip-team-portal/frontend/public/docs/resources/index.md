# Resources

**Route:** `/resources`

Shared reference data accessible across all modules.

## Sections

| Tab | Description |
|-----|-------------|
| Property Tables | 37 lookup tables for engineering data |
| Constants | Physical and mathematical constants |
| Documents | Google Drive integration and collections |
| Contacts | Internal and external contact directory |

## Property Tables

37 tables across 8 categories:
- Fasteners
- Mechanical
- Material
- Electrical
- Tolerances
- Finishes
- Process/Fluids
- Structural

Use `LOOKUP()` in equations to query table data.

## Constants

8 physical constants with shorthand syntax:
- `#CONST(g)` or `$g` - Standard gravity
- `#CONST(c)` or `$c` - Speed of light
- And more...

## Documents

Google Drive integration:
- OAuth2 connection
- File picker for Drive files
- Collections with color coding
- 8 document types supported

## Contacts

Contact directory with:
- Internal/external classification
- Expertise tagging
- Search and filter
