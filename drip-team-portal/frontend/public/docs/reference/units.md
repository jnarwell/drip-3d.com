# Units

The portal uses SI as base with support for common engineering units.

## Automatic Conversion

- Units convert automatically to match expected input
- Dimensional analysis validates compatibility
- Errors reported for incompatible units

## Dimensionless

Use `-` or leave blank for dimensionless quantities:
- Ratios
- Percentages
- Counts
- Coefficients (e.g., friction coefficient)
