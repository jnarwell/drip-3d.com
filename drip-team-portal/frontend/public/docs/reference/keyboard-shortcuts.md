# Keyboard Shortcuts

The portal supports a limited set of keyboard shortcuts.

## Equation Editor

| Shortcut | Action |
|----------|--------|
| Ctrl+, | Insert subscript |
| Ctrl+. | Insert superscript |

## General

| Shortcut | Action |
|----------|--------|
| Escape | Close modal or exit fullscreen |

## Expression Input

| Key | Action |
|-----|--------|
| Tab | Accept autocomplete suggestion |
| Enter | Submit value |
| Escape | Cancel editing |

## Standard Browser/OS

Standard shortcuts work in all text fields:

| Shortcut | Action |
|----------|--------|
| Ctrl/Cmd+A | Select all |
| Ctrl/Cmd+C | Copy |
| Ctrl/Cmd+V | Paste |
| Ctrl/Cmd+X | Cut |
| Ctrl/Cmd+Z | Undo |

## Notes

- Custom navigation shortcuts are not currently implemented
- Shortcuts work when appropriate input is focused
- Browser shortcuts may take precedence over app shortcuts
