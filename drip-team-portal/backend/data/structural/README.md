# Structural Engineering Data

This directory is reserved for structural engineering reference data.

## Planned Content

- **Beam sections** - Standard I-beam, W-beam, and channel dimensions
- **Structural steel grades** - ASTM A36, A992, A500 properties
- **Column buckling curves** - Euler/Johnson buckling data
- **Connection capacities** - Bolt/weld strength tables
- **Load combinations** - ASCE 7 load factor tables
